How to install Jekyll 3.x.x on Windows computer

us more!

This short guide will show you how to install Jekyll on Windows
There are several ways of doing this. However, I like the way suggested by David Burela on this post.
We use Chocolatey NuGet: Its a package manager, somewhat like apt-get in Linux.

Steps:
1. Open command prompt in admin mode(Windows key + x, a)
2. In the prompt type this:
@powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((new-object net.webclient).DownloadString('https://chocolatey.org/install.ps1'))" && SET PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin
That will install the package manager and set the PATH variables automatically.

3. Type: choco install ruby -y
This will install Ruby on your system.
4. Type: gem install jekyll
This installs jekyll on your system.
That's a pretty good way to install stuff if you ask me!
Those steps saved you a lot of clicking, typing and navigating in circles.
